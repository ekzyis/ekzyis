Welcome!

I am just a pleb who likes programming, cybersecurity and bitcoin
(chronological order).

I see this website as my personal playground where I can experiment with setups using minimal HTML and CSS (no javascript required here!), host a onion site, integrate protocols like L402 or nostr and whatever else I might come up with.

While doing this, I want to share my experiences with others in blog posts for the following reasons:

  • They might be useful as guides if others want to do similar things
  • Knowing that I can share whatever I learned with others later helps me stay motivated to keep learning more
  • I would love to talk more about topics that I am interested in with people (stacker.news made me realize this!)
  • I want to keep track of all the things I've learned over time
